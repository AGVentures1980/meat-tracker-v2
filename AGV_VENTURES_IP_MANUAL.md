## 1. SYSTEM OVERVIEW & FEATURES

Brasa Meat Intelligence is a complete operating system built specifically for the operational and financial complexities of "All-You-Can-Eat" Brazilian Steakhouses (Rodízio). Unlike generic restaurant systems, Brasa operates at the core of meat management, combating waste, ensuring end-to-end control, and automating data entry.

**CORE FEATURES:**

1. **Hybrid Rodízio Core Engine:** Precise calculation of LBS/Guest and Cost/Guest. Yield and Loss Rate analysis per meat cut. Dynamic pricing and analytical shift separation (Lunch vs. Dinner).
2. **The Garcia Rule & Accountability:** Rigorous monitoring of high-cost items ("Villains"). UI lockout obliges the team to report cold room status on time. Proactive overproduction detection.
3. **AI Prospecting Agent (Invoice Automation):** AI Agents working 24/7 connected to OneDrive. Automated reading of supplier PDFs (e.g., Sysco). Intelligent translation and mapping directly to the database. Zero human data entry.
4. **Offline-First PWA (Walk-In Cooler Checks):** Progressive Web App installable on mobile. "Weekly Pulse" can be done offline inside the cooler. Auto-syncs to the cloud when internet is restored.
5. **Smart Prep (Kitchen Intelligence):** Parametric daily prep projection based on history. Eliminates guesswork in meat scaling, reducing bottlenecks and avoiding excessive thawing of proteins.
6. **Network Management & Templates:** Master Ledger for real-time multi-unit management. Bulk Template application (e.g., "High Volume Weekend") with auto-adjusting targets for dozens of stores in seconds.
7. **Social Proof & Leaderboard:** Network cost leaderboard (Gold, Silver, Bronze). Gamification encouraging managers to optimize operations via healthy competition.
8. **Role-Based Access Control (RBAC):** Strict access levels (Master, Admin, Director, Manager). Immutable Audit Log for 100% traceability of all critical server changes.

## 2. TECHNICAL ARCHITECTURE & INNOVATION REPORT

**1. Executive Summary**
Brasa Meat Intelligence acts as a proactive financial supervisor, uniting non-linear continuous consumption mathematics (LBS/Guest) with Artificial Intelligence Agents for data ingestion and real-time waste auditing.

**2. Core Engine Modeling (Proprietary Algorithms)**
The backbone of AGV VENTURES' intellectual property lies in the mathematical calculation engines. The architecture rejects the traditional "Exact Recipe-Based Inventory" model and adopts the "Scalable Free Consumption" paradigm:
*   **LBS/Guest and Cost/Guest Calculation Algorithm:** Cross-references initial physical inventories, automated purchases, and closing counts against exact guest traffic.
*   **Yield & Loss Rate Matrix:** Creates detailed deboning/trimming profiles for each high-cost protein type to dynamically calculate if a unit's loss rate falls outside network tolerance.

**3. Autonomous Artificial Intelligence Integration**
*   **Monitoring Daemon (Watcher):** A native microservice continuously monitors designated OneDrive directories.
*   **Extraction Pipeline (OCR + LLM):** Reads unstructured supplier PDF content and processes it via Large Language Models.
*   **Nomenclature Normalization:** Translates supplier jargon (e.g., "BFF RIBEYE CHCE") into protected internal database categories.

**4. The Garcia Rule & Operational Accountability**
*   **Interface Lockout Protocol:** Restricts managerial functions until the team physically submits the count of sensitive cuts at the mandatory closing time.
*   **Untouchable Cryptographic Audit Log:** Every event is saved in the AuditLog stamped with server time, IP, and permission level, creating a "Single Source of Truth Ledger".

**5. Offline-First PWA Architecture**
*   Built using a Progressive Web App (PWA) architecture paired with Workbox/IndexedDB to allow staff to store inventory counts offline directly within the unit's browser buffer, syncing asynchronously when online.

**6. Smart Prep: Predictive Kitchen Forecasting**
*   Processes the moving average of recent weeks to generate a "Master Defrost Report" for butchers, shielding the company against the overproduction of highly perishable items.
